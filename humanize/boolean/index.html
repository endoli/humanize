Boolean Humanization

Parsing

For all languages, we support parsing "1" and "0" as true and false respectively.

In English, these lexical values map to true:

  • "ok"
  • "okay"
  • "on"
  • "true"
  • "yep"
  • "yes"

and these to false:

  • "false"
  • "no"
  • "nope"
  • "off"
